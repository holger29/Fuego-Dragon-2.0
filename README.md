La versión fuego-dragon-2.1 es la version donde se le integra la parte logica que consiste en integrar cogigo mysql y BD. Aqui se usa la gestion de base de datos,
el usuario en su pagina de perfil puede modificar sus datos personales, crear y enviar feedback. En la pagina del administrador: este puede gestionar a los usuarios, puede ver a todos los usuarios registrados en la 
BD y puede modificar tanto datos personales como restablecer contraseña, y tambien tiene el poder de eliminar a usuarios.
